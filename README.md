# Clone Exact de la Carte ANAE Algérie 🇩🇿

Ce projet est une réplique visuelle exacte de la page d'identité numérique de l'Agence Nationale de l'Auto-Entrepreneur (ANAE).

## 🛠️ Personnalisation Rapide

Ouvrez le fichier `index.html` et modifiez l'objet `userData` situé tout en haut du script :

- **cardNumber**: Votre numéro de carte (ex: 86832080).
- **address**: Votre adresse complète en majuscules.
- **expiryDate**: La date de validité.
- **username**: Votre nom d'utilisateur (après le @).
- **nameAr**: Votre nom complet en arabe.
- **photoUrl**: Le lien vers votre photo de profil.
- **activities**: Ajoutez ou supprimez des objets `{ title: "...", code: "..." }` pour mettre à jour vos domaines de compétence.

## 🚀 Déploiement

1. Créez un dépôt sur **GitHub**.
2. Téléchargez votre fichier `index.html` modifié.
3. Activez **GitHub Pages** dans les paramètres du dépôt.
4. Votre carte est en ligne !
